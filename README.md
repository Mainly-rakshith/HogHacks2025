# HogHacks 2025 Project

## Overview
This project is a Flutter application built during HogHacks 2025. It includes Dart code under the `lib` folder, dependency configuration in `pubspec.yaml`, and various resources under `assets`. It also contains backend configuration under the `supabase` directory.

## Main Components
- **lib**: Contains all Dart source code for widgets, screens, and logic.
- **assets**: Images and other static files used in the UI.
- **supabase**: Backend configuration for authentication and database.
- **pubspec.yaml**: Lists all Flutter dependencies and project metadata.
- **app.json / package.json**: Metadata and configuration for builds and dependencies.
- **.bolt / hooks**: Project scripts and environment-specific configuration.

## How to Run
1. Install Flutter SDK on your system.
2. Run:
   ```
   flutter pub get
   flutter run
   ```
3. Make sure `.env` has the correct Supabase keys.
